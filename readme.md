## Why use Avvio Editor?

Designing emails that work across all email platforms, browsers, etc is hard. **Avvio** is an opinionated editor that lets you craft emails in a hassle-free manner. We have a bunch of pre-designed components that make it easy for you to design any kind of email. Here is the list of components supported by **Avvio** with more being added.

- Logo
- Buttons and Variants
- Variables
- Text Formatting
- Image
- Alignment
- Divider
- Spacer
- Footer
- Inline Code
- Link Cards
- Section
- Columns
- Repeat
- Show If Condition
